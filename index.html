SMNG Typography

Body [Root:HTML]

The body/html is the root of design system. It's overrided by the official font-family and font-size as seen below

The official font is 'Open Sans' from Google Fonts

In base CSS file:

                
                    @import url('https://fonts.googleapis.com/css?family=Open+Sans:400,400i,600,600i');

CSS Snippet

                
                    /*Overrides*/
                    /*Larger Screens*/
                    html, body {
                        font-family: 'Open Sans';
                        font-size: 16px;
                    }
                    /*Regular or Smaller Screens*/
                    @media (max-width: 1366px) {
                        html, body {
                            font-size: 13px;
                        }
                    }

SMNG Buttons

Button

All buttons are direct children of the Bootstrap btn class and can consume other btn properties e.g. btn-lg, btn-sm etc.

However the btn class is overriden as seen below:

CSS Snippet:

                
                    /*Override*/
                    .btn {
                        margin-top: 0.25rem;
                        margin-bottom: 0.25rem;
                        cursor: pointer !important;
                    }
